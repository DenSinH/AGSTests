### MEMORY

AGS contains 9 memory tests. The names for these tests are also in the ROM, and are:
    - CPU EXTERNAL WORK RAM
    - CPU INTERNAL WORK RAM
    - PALETTE RAM
    - VRAM
    - OAM
    - CARTRIDGE TYPE FLAG
    - PREFETCH BUFFER
    - WAIT STATE WAIT CONTROL
    - CARTRIDGE RAM WAIT CONTROL
